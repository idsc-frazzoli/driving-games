## Docker installation and run

With Docker:

    $ make build
    
    $ make run

The results are going to be written in the `out-docker` dir.

## Without Docker

Requires Python >= 3.7.
    
Run manually the instructions in the Dockerfile.

Run using:

    $ dg-demo --reset
    
(The `--reset` option draws another sample after the first.)
